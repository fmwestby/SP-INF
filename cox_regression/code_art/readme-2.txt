*** Replication instructions ***
To replicate a figure or table, open the file name with the figure or table name and run it. The file will output figures as PDFs and tables as text files. After running all files you should see the following output files in your working directory.

*** Individual file descriptions ***
Below we provide specific descriptions and approximate run times for each R file. All files were run on an iMac with macOS High Sierra 10.13.4, a 4 GHz Intel Core i7 processor, and 32 GB of RAM. We used R version 3.4.1 ("Single Candle") and RStudio version 1.0.153.

rfunctions.R: Necessary R functions from the coxed package. Included to preserve these results as the package is updated.
figure1-figure2.R: Produces Figure1 and Figure2 (main text). [1 second]
figure3-table1-tableA4-figureA3a.R: Produces Figure 3 and Table 1 (main text), Table A4 row 1 (appendix), and Figure A3a (appendix). [52 seconds]
figureA1.R Produces Figure A1 (appendix). [28 minutes]
figureA2-tableA2.R: Produces Figure A2 and Table A2 (appendix). [47 seconds]
tableA1.R: Produces Table A1 (appendix). [12 seconds]
tableA3.R: Produces Table A3 (appendix). [41 minutes]
tableA4-figureA3b.R: Produces Table A4 row 2 (appendix) and Figure A3b (appendix). [55 seconds]
